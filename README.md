# 🎓 Student Data Organizer

## 📖 Description
Student Data Organizer is a Python-based menu-driven application that allows users to manage student records efficiently. It supports adding, displaying, updating, and deleting student data using collections like lists and dictionaries.

---

## 🚀 Features

### 📥 Add Students
- Enter student details:
  - ID
  - Name
  - Age
  - Grade (A+, A, B+, etc.)
  - Date of Birth
  - Subjects

### 📋 Display Students
- View all stored student records in a structured format

### ✏️ Update Students
- Modify existing student details using Student ID

### ❌ Delete Students
- Remove a specific student
- Option to delete all student records

### 📚 Subject Offer
- Displays available subjects:
  - Math
  - English
  - Science

### 🔁 Menu Driven System
- Continuous execution using loop until user exits

---

## 🧑‍💻 Technologies Used
- Python
  - Lists & Dictionaries
  - Loops (`while`, `for`)
  - Conditional Statements (`if-elif-else`)
  - Input Validation (`isdigit()`, `isalpha()`)
